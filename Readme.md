## Getting Started
.
To get a local copy up and running the project you can follow the next steps!

### Prerequisites

To run this project first you need to install node.js and express.js.
If you follow the instructions presented in the next section you gonna be more than fine and be able to run an amazing API locally in your machine üëç

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dudustri/NodeAPI
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install Express
   ```sh
   npm install express
   ```
4. Run the node using the terminal
   ```sh
   npm start
   ```
5. Have fun! Use the API locally üòé
